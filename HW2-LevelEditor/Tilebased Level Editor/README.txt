Controls
--------------

Tile Settings
'up' Key - Changes currently selected tile (up through list)
'down' Key - Changes currently selected tile (down through list)

Drawing
Left Mouse - Draw Tile
Right Mouse - Erase Tile

Saving and Loading
'S' Key - Save Level (to level.txt)
'L' Key - Load Level (from level.txt)

MISC
'Spacebar' - Take a screenshot
